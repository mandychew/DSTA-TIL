class tilsdk.SignedDistanceGrid(width=0, height=0, grid=None, scale=1.0)

Grid map representation.

Grid elements are square and represented by a float. Value indicates distance from nearest obstacle. Value <= 0 indicates occupied, > 0 indicates passable.

Grid is centered-aligned, i.e. real-world postion corresponds to center of grid square.

Parameters

  • width (int) – Width of map in number of grid elements, corresponding to real-world x-axis. Ignored if grid parameter is specified.

  • height (int) – Height of map in number of grid elements, corresponding to real-world y-axis. Ignored if grid parameter is specified.

  • grid (nxm ArrayLike) – Numpy array of grid data, corresponding to a grid of width m and heigh n.

  • scale (float) – Ratio of real-world unit to grid unit.
